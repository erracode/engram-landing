# Welcome to Engram Docs

**Engram** is a persistent memory system for AI coding agents. It enables agents to remember decisions, patterns, and context across sessions.

## Quick Links

- [Installation](/docs/installation) - Set up Engram on your system
- [Agent Setup](/docs/agent-setup) - Configure Engram for your AI agent
- [Plugins](/docs/plugins) - Install OpenCode and Claude Code plugins
- [Architecture](/docs/architecture) - System design and structure
- [Skills](/docs/skills/) - Agent Skills reference guide

## Getting Started

1. **Install Engram** following the [installation guide](/docs/installation)
2. **Set up your agent** with the [agent setup](/docs/agent-setup) instructions
3. **Install plugins** for your preferred IDE or editor
4. **Read the documentation** to understand how Engram works

## Features

- **Persistent Memory**: Remember context across sessions
- **Agent Skills**: Specialized knowledge for specific tasks
- **Git Sync**: Automatic synchronization with Git repositories
- **MCP Tools**: Model Context Protocol integration
- **Terminal UI**: Interactive command-line interface
