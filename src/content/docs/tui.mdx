## Screens

### Dashboard Screen

The main dashboard shows:
- **Recent memories** — Last 20 observations with timestamps
- **Session summaries** — Active and completed sessions
- **Quick stats** — Total memories, projects, agents
- **Recent activity** — Timeline of memory operations

The dashboard is the default view when you launch `engram tui`.

### Detail Screen

Drill into any memory to see:
- **Full content** — Complete observation text
- **Metadata** — Title, type, topic key, revision count
- **Timeline** — Related observations from same session
- **Actions** — Update, delete, or export the memory

### Search Screen

Full-text search across all memories:
- **FTS5 search** — Natural language queries
- **Filters** — By project, scope, type, date range
- **Results** — Compact list with IDs for quick navigation
- **Drill in** — Click any result to see full content

### Timeline Screen

Chronological context around a specific observation:
- **Session context** — What happened before/after
- **Related memories** — Same topic or project
- **Timeline view** — Linear view of all operations

### Settings Screen

Configure TUI preferences:
- **Theme** — Catppuccin Mocha (default)
- **Default project** — Filter all views to project
- **Auto-refresh** — Enable/disable real-time updates
- **Display options** — Show/hide metadata, dates, etc.

## Usage Examples

### Save a Memory

From the TUI, press `s` to quickly save a new memory:

```
Title: Fixed N+1 query in user list
Type: bugfix
Content:
What: User list was making N+1 database queries
Why: Performance issue causing slow page loads
Where: src/internal/store/user.go:142
Learned: Use Include() for eager loading in GORM
```

The TUI validates the format and saves to SQLite with FTS5 indexing.

### Search Memories

Press `/` to search:

```
> auth middleware
```

The TUI shows matching results with:
- Title
- Type
- Date created
- Project name

### Export Memories

Press `e` to export all memories:

```
Export memories to JSON? [y/N]
```

The TUI exports all observations (including deleted) to a JSON file in the current directory.

### Sync with Git

Press `g` to sync memories:

```
Syncing memories to .engram/...
✓ Exported 12 new memories
✓ Created .engram/manifest.json
```

Then commit to git:

```bash
git add .engram/ && git commit -m "sync engram memories"
```

## Custom Themes

You can customize the TUI colors by editing the style configuration in `internal/tui/styles.go`:

```go
var (
  base    = color.New(color.WithColorScheme(color.SchemeCatppuccinMocha))
  accent  = color.New(color.FgCyan)
  warning = color.New(color.FgYellow)
  danger  = color.New(color.FgRed)
)
```

Create your own theme by defining new colors and applying them to specific elements.

## Tips

- **Quick save** — Press `s` from any screen to quickly save a memory
- **Bulk delete** — Use `Space` to select multiple items, then `x` to delete
- **Advanced search** — Use `AND`, `OR`, `NOT` operators in search queries
- **Project filter** — Set default project in settings to filter all views
- **Memory Protocol** — The TUI shows the Memory Protocol reminders at startup
