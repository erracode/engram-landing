# 🚀 Engram Landing Page & Documentation

## Two Separate Applications

### 1. Landing Page (Simulator)
**Location**: Current directory (`engram-landing/`)
**Port**: 4321

```bash
# In this directory
npm run dev
```
👉 Visit: **http://localhost:4321/**

Features:
- 3D Isometric Diorama
- Engram Simulator
- Window Management System
- TopBar Navigation
- MDX Documentation Windows

---

### 2. Starlight Documentation
**Location**: `docs-src/` directory
**Port**: 4323

```bash
# Change to docs directory
cd docs-src

# Install and run
npm install
npm run dev
```
👉 Visit: **http://localhost:4323/**

Features:
- Professional documentation site
- Sidebar navigation
- 15 documentation pages
- Search & navigation
- Nothing Design styling
- Agent Skills index
- Full Engram repo docs

---

## Quick Start

**Terminal 1** (Landing Page):
```bash
npm run dev
```

**Terminal 2** (Documentation):
```bash
cd docs-src
npm install
npm run dev
```

Both run simultaneously! 🎉
